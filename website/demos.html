Governance

Filesystem Governance

An agent writes malicious files, modifies configs, deletes critical data. But the host stays completely untouched. Diff shows every change. Rollback discards everything instantly.

83sCOW overlaydiff / rollback

Credential Vault

Store API keys encrypted with ChaCha20-Poly1305. The agent sees masked values. Real keys are injected at runtime but never appear in env vars, config files, or shell history.

57sEncrypted secrets

Network Isolation

Per-pod DNS whitelist in action. Allowed domains resolve normally. Blocked domains return NXDOMAIN. Every DNS query is logged to the audit trail with timestamps.
