# Third-Party Licenses

envpod is built on open-source Rust crates. All 289 dependencies use
permissive licenses fully compatible with BSL 1.1. No copyleft (GPL,
AGPL, MPL, SSPL) dependencies exist.

The complete license texts for every dependency are in
[THIRD-PARTY-LICENSES.html](THIRD-PARTY-LICENSES.html), generated by
[cargo-about](https://github.com/EmbarkStudios/cargo-about).

To regenerate:

```bash
cargo about generate about.hbs > THIRD-PARTY-LICENSES.html
```
