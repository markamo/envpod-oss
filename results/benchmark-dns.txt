  Key insight:
    Docker/Podman: unfiltered DNS passthrough — no governance
    Envpod:        whitelist-filtered DNS + query logging + isolated network
    Envpod is faster WITH governance than Docker/Podman WITHOUT it.
    The in-container loop proves the overhead is in container entry,
    not in envpod's DNS filtering or network stack.

  What envpod adds (zero extra cost):
    + Per-pod DNS whitelist filtering with query logging
    + Anti-DNS-tunneling detection
    + COW filesystem (diff/commit/rollback)
    + Action-level audit trail (JSONL)
    + seccomp-BPF syscall filtering
    + Credential vault
